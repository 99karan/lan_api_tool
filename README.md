# 🚀 Flutter LAN API Tool

A cross-device API testing tool built with Flutter that works over local network (LAN).

## ✨ Features

* 📡 Send HTTP requests (GET, POST, PUT, DELETE)
* 📦 Save request/response history
* 🌐 Discover devices over LAN (mDNS)
* 🔌 Sync requests across devices via TCP sockets

## 🏗️ Architecture

* Flutter Desktop (macOS / Windows)
* Dio (HTTP client)
* Hive (local storage)
* Riverpod (state management)
* mDNS (device discovery)
* TCP sockets (real-time sync)

## 📌 Roadmap

* [ ] Phase 1: API client + local storage
* [ ] Phase 2: LAN discovery
* [ ] Phase 3: Real-time sync

flutter pub get
